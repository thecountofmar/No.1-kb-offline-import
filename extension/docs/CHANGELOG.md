# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [1.1.0] - 2026-07-10

### 新增
- 点扩展图标直接下载，无需弹窗确认
- 标题提取支持 `<h1>`、`og:title` 等多种来源
- 右键菜单「📥 智能抓取此页」

### 优化
- 文件名清理逻辑：合并连续下划线，空标题用时间戳兜底
- 下载改为页面内 `Blob + a.click()`，文件名 100% 正确

### 移除
- 删除 `popup.html` 和 `popup.js`（不再需要弹窗）
- 移除 `downloads` 权限（下载在页面内完成）

## [1.0.0] - 2026-07-10

### 初始版本
- Manifest V3 架构
- 智能抓取网页正文为 Markdown
- 支持微信公众号、任意网页
- 正文提取：自动识别 `#js_content` / `article` / `main` 等容器
- 噪音清洗：去除导航、广告、侧边栏、相关推荐、评论区
- 段落格式保留：标题、段落、列表、引用、代码块
- Markdown 输出带 frontmatter 元信息
